Getting Started with HTML & CSS: A Complete Beginner's Guide

John Doe January 15, 2024 5 min read

Introduction to Web Development

HTML (HyperText Markup Language) and CSS (Cascading Style Sheets) are the fundamental building blocks of the web. HTML provides the structure of a webpage, while CSS handles the visual presentation. Together, they allow you to create beautiful, functional websites.

What is HTML?

HTML is the standard markup language for documents designed to be displayed in a web browser. It uses a system of tags and attributes to define the structure and content of web pages.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First Webpage</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first webpage.</p>
</body>
</html>

What is CSS?

CSS is a style sheet language used for describing the presentation of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    color: #333;
}

h1 {
    color: #0066cc;
    text-align: center;
}

Basic HTML Structure

Every HTML document follows a basic structure:

  • <!DOCTYPE html> - Defines the document type
  • <html> - The root element
  • <head> - Contains meta information
  • <body> - Contains the visible content

Common HTML Elements

Element Description Example
<h1> to <h6> Heading tags <h1>Main Title</h1>
<p> Paragraph <p>Some text</p>
<a> Link <a href="#">Link</a>
<img> Image <img src="image.jpg" alt="Description">

CSS Selectors and Properties

CSS uses selectors to target HTML elements and apply styles. Here are the most common selectors:

Element Selector

p {
    color: blue;
    font-size: 16px;
}

Class Selector

.highlight {
    background-color: yellow;
}

ID Selector

#header {
    background-color: #333;
    color: white;
}

Putting It All Together

Here's a complete example of HTML with CSS:

HTML File (index.html)

<!DOCTYPE html>
<html lang="en">
<head>
    <link rel="stylesheet" href="styles.css">
    <title>My Website</title>
</head>
<body>
    <header class="site-header">
        <h1>Welcome to My Site</h1>
    </header>
    <main>
        <p class="intro">This is my first website!</p>
    </main>
</body>
</html>

CSS File (styles.css)

.site-header {
    background-color: #4CAF50;
    color: white;
    padding: 20px;
    text-align: center;
}

.intro {
    font-size: 18px;
    line-height: 1.6;
    color: #333;
}

Next Steps

Now that you understand the basics of HTML and CSS, here's what you can do next:

  1. Practice by creating simple web pages
  2. Learn about responsive design with media queries
  3. Explore CSS frameworks like Bootstrap
  4. Move on to JavaScript for interactivity
